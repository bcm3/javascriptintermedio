¿Qué es y cómo funciona el hoisting?
- Es cuando el interprete interno de JS pasa a ubicar las variables al inicio.

¿Qué es una “temporal dead zone”?
- Cuando mostramos por consola una variable vacia, y la inicializamos despues.

¿Qué diferencia hay entre var y let?
var declara una variable de scope global o local para la función sin importar el ámbito de bloque. Permite hoisting.
let declara una variable de scope global, local para la función o de bloque. Es reasignable y NO permite hoisting.
